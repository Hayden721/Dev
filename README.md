## 📌 기능
### 일반 사용자 페이지
회원 관리
- 로그인
- 회원가입
- 로그아웃
- 회원 수정
- 비밀번호 찾기
  
예약
- 공간 예약
- 예약 취소
- 예약 조회
  
###  공간 판매자(seller) 페이지
회원 관리
- 로그인
- 회원 가입
- 로그아웃
- 비밀 번호 찾기

공간 관리
- 공간 조회
- 공간 추가
- 공간 삭제
- 공간 수정

예약 관리
- 예약 조회
- 예약 취소
