# Project
공간 예약 서비스
일반 사용자와 공간 제공자가 사용 관리할 수 있는 서비스를 개발합니다.



###  일반 사용자 페이지
사용자가 사용, 예약하고자 하는 유형의 공간을 날짜와 시간을 지정하여 예약할 수 있고 예약 조회, 관리 등을 할 수 있습니다.



###  공간 제공자(seller) 페이지
제공하는 공간을 등록, 관리 등을 할 수 있습니다.
